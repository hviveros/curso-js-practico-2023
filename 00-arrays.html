Qué es un array? Es una lista de elementos const array = [1, 'jaja', true, false]; Qué es un objeto? Es una de elementos PERO cada elemento tiene un nombre clave const obj = { nombre: 'fulano', edad: 22, comidasFavoritas: [ 'Pollo', 'Vegetales' ] } Cuándo es mejor usar objetos o arrays? Arrays cuando lo que haremos en un elemento es lo mismo que en todos los demás (la regla se puede incumplir) Objetos cuando los nombres de cada elemento son importantes para el algoritmo Puedo mezclar arrays con objetos o incluso objetos con arrays? Sí, los arrays pueden guardar objetos. Y los objetos pueden guardar arrays entre sus propiedades
